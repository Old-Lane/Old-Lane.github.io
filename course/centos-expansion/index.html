CentOS 7根目录在/dev/sda下扩容

针对 VMware 虚拟机默认分区的 CentOS 7 系统，根目录在 /dev/sda

原理：先删除 sda3 ，再重建 sda3

参考：https://blog.csdn.net/yang1393214887/article/details/120345884

# 开始扩容

# 1、VMware 虚拟机扩容

VMware -> 编辑虚拟机设置 -> 扩展

# 2、CentOS 扩容

  1. 进入 CentOS 系统

  2. 查看分区状态

  3. 删除 sda3 然后重建

  4. 刷新： resize2fs /dev/sda3

    执行 cat /etc/fstab 查看系统文件格式

    CentOS7 默认 xfs 格式，执行 xfs_growfs /dev/sda3

    红帽是 ext 格式，执行 resize2fs /dev/sda3

  5. 重启： reboot
